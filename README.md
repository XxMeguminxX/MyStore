# MyStore

Welcome to MyStore, your one-stop shop for all your shopping needs!

## Features
- User-friendly interface
- Secure payment options
- Wide range of products

## Installation
To install MyStore, follow these steps:
1. Clone the repository: `git clone https://github.com/XxMeguminxX/MyStore.git`
2. Navigate to the project directory: `cd MyStore`
3. Install dependencies: `npm install`

## Usage
To start the application, run:
```bash
npm start
```
